# Historia de Usuario: Visualización de cursos disponibles

**ID:** `HU-002` | **Prioridad:** `Alta` | **Estimación:** `M`

## Descripción
**Como** usuario  
**Quiero** visualizar los cursos disponibles  
**Para** conocer las opciones de formación ofrecidas en la plataforma

## Criterios de Aceptación

### Escenario 1: Consulta de cursos
- **Dado** que el usuario accede a la sección de cursos
- **Cuando** el sistema obtiene la información disponible
- **Entonces** se muestra un listado con los cursos registrados

### Escenario 2: Error al cargar cursos
- **Dado** que el usuario accede a la sección de cursos
- **Cuando** ocurre un error en la obtención de datos
- **Entonces** el sistema muestra un mensaje de error

### Escenario 3: Visualización de información del curso
- **Dado** existen cursos registrados
- **Cuando** el usuario visualiza el listado
- **Entonces** el sistema muestra título, descripción, instructor y duración del curso

## Notas
- Los datos serán consumidos desde JSON Server.
- Endpoint mock sugerido: `/courses`
- Se recomienda diseño responsive para dispositivos móviles.
